# Branding Assets

This directory contains branding assets for FluStudio.

## Required Icons

Place the following icon files in the `icons/` directory:

1. **app_icon.ico** - Application icon for Windows (recommended: 256x256 or 512x512)
2. **app_icon.png** - Application icon for other platforms (recommended: 256x256 or 512x512)
3. **logo.png** - Logo for About dialog (recommended: 128x128 or 256x256)

## Icon Specifications

- **Format**: ICO for Windows, PNG for cross-platform
- **Size**: 256x256 or 512x512 pixels recommended
- **Background**: Transparent or solid color
- **Style**: Should represent Flutter/development theme

## Creating Icons

You can create icons using:
- Online icon generators
- Image editing software (GIMP, Photoshop, etc.)
- Icon design tools (IconWorkshop, etc.)

## Current Status

Icons are optional. If not provided, the application will work without them, but window icon and About dialog branding will not be displayed.
